# ios-goggleAuth
A user signup and logout process using google-authentication

### Note ⚙️
The application was built using: 
* Xcode 12.5.1
* Swift 5.4
* macOS Big Sur 11.5.2

### Core Technologies 📲
* Swift
* Storyboards
* GoogleSignIn

### To run
* clone project
* open folder path in terminal. run `pod install`
* run the `xcworkspace` project.
